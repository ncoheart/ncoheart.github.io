关于HEXO安装失败的解决方法

2020年03月20日 141 字 大概 1 分钟

目前国内npm源有问题；所以键入如下代码即可安装成功:

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install hexo-cli -g

如果报错如下：

cnpm : 无法加载文件 C:\Users\ncoheart\AppData\Roaming\npm\cnpm.ps1，因为在此系统上禁止运行脚本。有关详细信息，请参阅 ht
tps:/go.microsoft.com/fwlink/?LinkID=135170 中的 about_Execution_Policies。
所在位置 行:1 字符: 1
+ cnpm install hexo-cli -g
+ ~~~~
    + CategoryInfo          : SecurityError: (:) []，PSSecurityException
    + FullyQualifiedErrorId : UnauthorizedAccess

运行powershell(管理员)，输入一下代码：

set-ExecutionPolicy RemoteSigned

然后选A，继续上面的操作
